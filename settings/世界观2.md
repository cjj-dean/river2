# 《扶摇界：破妄之主》世界观设定（综合重构版）

基于最新的卡片设定，本作的世界观建立在一个“天道残缺、神明陨落、飞升即死亡”的残酷黑暗修真世界之上。

## 一、 世界核心背景与暗黑逻辑

### 1. [[陨神大劫]]与[[绝天地通]]
远古时期，先天神魔“弑天”攻打天道具象化的【[[天衡]]】，导致诸天破碎，史称**“[[陨神大劫]]”**。[[天衡]]碎裂成无数【[[大道残片]]】散落人间，化作洞天福地与绝世功法的源头。
为保人族火种，上古圣皇斩断连接天地的“[[建木]]”，史称**“[[绝天地通]]”**。自此真仙绝迹，神明不显，灵气随着[[灵脉]]枯竭进入慢性死亡的末法时代。

### 2. [[天道收割]]与飞升骗局
这是本世界最大的谎言。修士达到上三境巅峰后的“白日飞升”，实则前往的根本不是仙界，而是被天道“回收”肉身、神魂和修为，转化为天地本源以反哺残缺的天道。
那些试图抗拒飞升、超脱天道的大能，最终也逃不过【[[天人五衰]]】，只能被迫选择【[[兵解]]】（主动解体飞升）或【[[化道]]】（融入天地法则），无一不是被残缺天道吃干抹净。

### 3. [[守岁人]]与[[血祭]]苍生
[[建木]]被斩断后，主干并未腐朽，而是沉入[[归墟]]之底。一群名为【[[守岁人]]】的顶尖大能（如[[太一玄宗]][[老天师]][[赵黄庭]]）将[[建木]]主干制成【[[撑天木]]】，插入地心支撑残缺的天道框架，防止世界立刻崩塌。
但[[撑天木]]需要用生灵精血浇灌才能延缓腐朽。[[守岁人]]与[[大奉仙朝]]皇帝为了“续命”与“长生”，暗中发动黑暗动乱，用天下无数凡人的命进行【[[血祭]]】，形成了高位者吃底层人的残酷维稳逻辑。

### 4. [[浊气]]、[[煞气]]与[[业障]]
- **[[浊气]]**：天地灵气混杂了远古神魔的怨念。吸纳过度或[[道心]]不坚，修士极易【[[走火入魔]]】，变成杀戮怪物。
- **[[煞气]]**：战争与杀戮产生的负面能量，能污染灵气。
- **[[业障]]与[[天刑]]**：修士若屠戮凡俗、违背法则会沾染极重[[业障]]，引来绝对抹杀的【[[天刑]]】或[[雷劫]]。唯有缴纳天道坍缩形成的【[[岁月铢]]】方可免除一死。

## 二、 力量体系与修真流派

### 1. [[衍灵道]]（练气士）
主流修真体系，讲究“窃取天机，将自身化为太初仪轨”。极其依赖灵气、法宝与资源，手段繁多，杀伤范围极大，但肉身孱弱。
- **下三境（塑基）**：[[凝窍境]]（开辟气窍） → [[观骨境]]（灵气刻骨） → [[叩关境]]（寻得大道真意）
- **中三境（立道）**：[[筑鼎境]]（丹田铸鼎，御剑飞行） → [[寻泉境]]（融入山水，改变地形） → 某未知瓶颈
- **上三境（夺理）**：[[玉枢境]] → [[忘筌境]]（言出法随，篡改物理法则） → [[天命境]]（上三境之巅）

### 2. [[镇渊武夫]]（纯粹武道）
凡人逆袭之路。不吸纳任何天地灵气，全凭压榨肉身极限，养一口不屈真气。同阶单挑无敌，十步之内可瞬间打爆练气士，但寿命极短，只有突破至后三重方可延寿。
- **前三重（熬体）**：[[泥胎境]]（刀枪不入） → [[铁骨境]]（骨硬如铁） → [[铜血境]]（气血如炉震碎阴邪）
- **中三重（断江）**：[[摧城境]]（拳意摧城） → [[断江境]]（打断大江） → [[镇海境]]（镇压海啸）
- **后三重（破天）**：[[撼星境]]（打碎星辰） → [[无漏境]]（肉身无漏万法不侵） → [[武神境]]（击碎[[天道刻痕]]）

### 3. [[经纬道]]（儒家）
代天立言，教化众生。不需打坐，通过读书明理养胸中“浩然正气”，专克妖邪。
- **境界划分**：[[养气境]] → [[诚意境]] → [[立规矩境]]
- **特点**：只要道理吃透可一夜破境；一旦言行不一，境界瞬间跌落甚至反噬。

### 4. [[妖族]]体系
体蕴远古神魔[[浊气]]，每次突破必遭恐怖[[雷劫]]。
- **境界划分**：[[启智期]] → [[炼骨期]] → [[结丹期]]（引三九[[雷劫]]） → [[化形期]]（引六九[[雷劫]]，匹敌[[玉枢境]]） → [[妖王]]（觉醒[[本命神通]]）

### 5. 鬼修体系
吸纳阴气与古战场[[煞气]]，惧怕阳光、浩然气与雷法。
- **境界划分**：[[游魂]] → [[凝煞]]（产生幻术） → [[塑阴体]] → [[鬼仙]]（度纯阳[[雷劫]]，阴极生阳）

## 三、 世界地理版图

天下分为四大洲，被充斥空间乱流的【[[叹息之海]]】分割，深海极底则是万物终焉的【[[归墟]]】。

### 1. [[中土神洲]]（首善之洲）
灵气最充裕的核心，[[大奉仙朝]]统治的腹心地带。
- **[[龙城]]**：大奉都城，[[九州龙脉]]汇聚之地。布有道阶残阵【[[九州龙气护城阵]]】，严禁修士御空飞行。
- **[[太衍山脉]]**：天下灵气祖脉，主峰高达九万丈，【[[太一玄宗]]】总部所在，地底藏有[[撑天木]]。
- **其他名地**：[[八百里龙原]]、[[长安道]]、[[中州城]]（商贸）、[[灵茶城]]（[[白鹿书院]]驻地）、[[机城]]（[[千机阁]]驻地）。

### 2. [[北冥霜洲]]（极寒武境）
终年冰雪覆盖，武夫与[[妖族]]厮杀的极北苦寒之地。
- **[[极寒武帝城]]**：立于【[[截云长城]]】之上，纯粹武夫的大本营，由城主[[铁浮屠]]镇守，抵御妖兽南下。
- **[[十万大山]]**：远古洪荒禁地，毒瘴弥漫，【[[万妖庭]]】的[[妖族]]祖地，十二大[[妖王]]盘踞于此。
- **[[凛冬冰河]]**：半冰封的天堑水脉。

### 3. [[东临剑洲]]（剑修圣地）
气候温润，常年剑气纵横。
- **[[葬剑海]]**：方圆万里的远古天坑，常年剑气化雨，下三境踏入瞬间会被绞成肉泥。
- **[[洗剑池]]**：位于[[葬剑海]]旁，天下剑修圣地。
- **其他名地**：[[蜀山山脉]]、[[一线天崖]]、[[怒涛江]]、[[铸剑谷]]。

### 4. [[南疆瘴洲]]（毒蛊迷沼）
湿热多雨，由无数岛屿和迷雾海域组成，盛产阴属性与毒属性[[灵砂]]。
- **毒瘴林**：医毒圣地【[[百草谷]]】所在地。
- **[[蛊师岛]]**：十二[[蛊师部落]]聚集地。
- **其他势力**：黄泉沼（[[赶尸客栈]]）、[[暗渊]]（刺客组织）、[[鬼市]]。

### 5. [[西极荒漠]]（失落佛国）
上古佛国遗址，黄沙万里，昼夜温差极大。
- **[[须弥圣山]]**：荒漠中的千里雪山，佛门圣地【[[须弥寺]]】所在。
- **血海魔窟**：魔道禁地，【[[血海魔宗]]】所在，与[[须弥寺]]展开了千年的佛魔之争。

## 四、 天下顶尖势力格局

1. **[[大奉仙朝]]**（[[天命境]]级）：统治[[中土神洲]]八成疆域的至尊王朝。皇帝拥有【[[传国玉玺]]】，借皇道[[龙气]]可力敌大能。麾下有[[镇抚司]]（冷血鹰犬，统领[[魏长渊]]）、[[神枢营]]（皇家禁军，统领[[萧红叶]]）、[[打更人]]等。
2. **[[太一玄宗]]**（[[天命境]]级）：道教正统，天下第一大宗。[[老天师]][[赵黄庭]]为幕后执棋者。
3. **[[白鹿书院]]**（上三境级）：儒家圣地，半圣[[颜回风]]看穿天道阴谋，暗中布局对抗收割。
4. **[[洗剑池]]**（[[天命境]]级）：只敬手中剑的剑道霸主，信奉“[[无瑕剑心]]”与一剑破万法。
5. **[[万妖庭]]**（上三境级）：大荒遗种（[[猿魔]]、九尾天狐[[涂山氏]]等）组建的[[妖族]]联盟，图谋南下。
6. **[[千机阁]]**：偃师传承，信奉“血肉苦弱，机关飞升”，首席偃师[[墨输]]将自身彻底机械化。
7. **[[百草谷]]**：从悬壶济世转变为手段残忍的生化毒宗，谷主[[药师琉璃]]手持[[生杀血葫芦]]。
8. **[[跨洲渡船]] & [[天机阁]]**：游离于正邪之外的中立巨头，阁主[[晏三笑]]与[[红姑]]掌控天下情报与跨洲垄断贸易，只认[[青玉钱]]。

## 五、 经济、资源与特殊法则

### 1. [[灵脉]]与货币体系
- **[[天品灵脉]]**（天下仅10条）：修行倍率100倍。年产1枚【**[[岁月铢]]**】（因果通货，生锈古铜钱，大能死后坍缩凝结，可抵消[[天刑]]或买断因果）。
- **[[地品灵脉]]**（约100条）：年产【**[[青玉钱]]**】（中级流通货币）。
- **玄/黄品[[灵脉]]**：年产【**[[灵砂]]**】（发光颗粒，底层修士的命脉）。
- **[[凡俗金银]]**：凡人使用，练气士极度排斥，认为其含有“红尘[[浊气]]”。

### 2. 特殊法则
- **[[拾道术]]**：扶摇界术法非凭空创造，而是从[[大道残片]]中感悟，因此皆有残缺限制，过度使用会遭法则冲突或[[浊气]]反噬。
- **[[香火供奉]]**：凡人信仰凝聚的人道法则。山神城隍需得王朝或宗门敕封，一旦香火断绝，神灵即陨落。
- **[[破妄之眼]]**：主角[[宁观]]融合的“[[天衡]]核心残片”。赋予其【洞微】（看破破绽）、【斩络】（切断灵气连接）、【窥命】（观测因果）的逆天能力，也是天道诱导其成为“修补工具”的诱饵，但主角将以此看破虚妄，重拳砸碎这吃人的天道磨盘。
