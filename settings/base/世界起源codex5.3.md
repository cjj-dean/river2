# 扶摇界：世界起源

太初之时，无天无地，无岁月可记。上下不分，清浊同沦，万象皆伏于一团沉黯元炁。彼时既无生灵，亦无“死”之一说；既无大道，亦无“逆道”可言。天地只是沉睡，不曾醒来。

直到某一瞬，沉黯中生出第一缕“明”。
那“明”并非光，而是一道“可被遵循”的秩序。后世称之为“太一”。

太一既生，便有分判。
轻者上腾，为清；重者下坠，为浊。
清炁舒展成天纲，浊炁凝结为地骨。星辰、海陆、寒暑、昼夜，皆由此开端。然太一知晓：只分天地，不足以久安；若无规矩，万物终归乱灭。于是太一以自身为契，化入天纲，立下第一重法度——“凡生者有尽，凡得者有偿，凡逆者有报”。

自此，扶摇界有了可行之理，也有了不可越之限。

太一化道之后，天地虽成，仍未稳固。清浊交界处裂出无数“玄罅”，罅中生出先天神魔。其神者近清，持衡守序；其魔者近浊，噬界逐裂。神魔本非善恶之别，不过立场各殊：一方欲固天纲，一方欲还太初。两道相争，遂起古战。此战延绵不可纪年，山海翻覆，星斗坠落，地脉多断，九幽由是成形。

大战末尾，神魔俱残，天纲亦伤。
太一所化之道上，终于出现第一道“缺”——后世称“天缺”。

天缺不在山川之间，而在规则本身。它如丝缝，细不可见，却昼夜渗出浊息。浊息入海，海生黑潮；浊息入土，土化绝煞；浊息入心，心起妄相。故扶摇界自古有“盛极而衰”之循环：每逢浊潮大作，王朝崩坏，宗门易主，旧秩序灰飞烟灭；待清炁回升，又有新法新国再起。世间兴亡，不过清浊消长的一次次回响。

先天神魔退场后，后天生灵登上世局。
凡人、异种、山泽精怪，皆不得天生伟力，只能以“小天地”叩问“大天地”：

- 以命炉炼气血，对抗寂灭；
- 以镇心定神识，压伏妄相；
- 以掌理立道心，辨真伪；
- 以有情守人性，抵无情之神性同化。

故修行并非拾级而上，而是时时与代价对坐。你借天一寸，天收你一尺；你赢一场劫，命里便添一重债。所谓天骄，不过是更早看见账本的人；所谓大能，不过是还债还到最后仍能提剑的人。

此界最深之矛盾，由起源那一刻便已注定：
若彻底归顺天纲，可得长久与强绝，却将渐失悲喜，化作规则的器皿；
若执意护持自我，可留一口真心与自由，却必经劫火重重，稍失便坠浊海。

于是，扶摇界万古同问，至今无解：
“人成道，究竟是为了更像天，还是为了不失为人？”

这，便是扶摇界的起源，也是后来一切战祸、王朝、宗门、爱恨与背叛的根。
天地从不偏爱谁，它只记得两件事：
清浊未止，因果未平。
