# 扶摇界地理体系

## 一、总纲

扶摇界的地理，不是普通的“大陆地图”，而是一个同时存在**水平疆域、垂直层次、折叠洞天、裂隙副本**的多维世界。

要理解扶摇界地图，必须同时看四个维度：

1. **平面地图**：王朝、宗门、山海、边荒如何分布。
2. **垂直地图**：九天、人间、九幽之间的高低结构。
3. **折叠地图**：洞天、福地、秘境、遗府这类“内外不一致”的空间。
4. **裂隙地图**：天缺支脉、古战场遗痕、规则坏死区。

## 二、平面疆域

扶摇界地表可大致分为六大板块。

### 1. 中州腹地

- 特点：地脉最稳、承运最盛、人口最密。
- 势力：大王朝、中州大宗、社稷神庭主系。
- 功能：天下秩序中心。

这里不是最危险的地方，但一定是最复杂的地方。所有天下级规则，最后都要落到这里执行。

### 2. 四极边疆

- 特点：靠近裂伤、绝煞、妖域或古战场。
- 势力：边军、军镇宗门、流民寨、妖部、水府。
- 功能：主线最适合开局的区域。

边疆是秩序最脆弱、真相最容易漏出来的地方，也是英雄和疯子最多的地方。

### 3. 山海古域

- 特点：古老、辽阔、自然法则更偏异族。
- 势力：妖盟、真灵遗脉、山泽精怪、水府王庭。
- 功能：异族政治与古血复苏主场。

### 4. 大泽水脉区

- 特点：河网、湖泽、水府和地下水脉交错。
- 势力：水府龙属、航运世家、漕运王朝、江河神系。
- 功能：贸易、走私、信息流动中心。

### 5. 废土绝地带

- 特点：绝煞浓厚、地脉破碎、灾变频发。
- 势力：少量镇守据点、掘宝客、流亡者、污染群落。
- 功能：副本密度最高，资源与死亡并存。

### 6. 古战场沉区

- 特点：上古神魔大战遗痕最深。
- 势力：几乎无常规政权，只有零散守墓者、掘史者、诡种群落。
- 功能：真相线和高危秘境核心来源。

## 三、垂直世界结构

### 1. 九天之上

- 能量特征：大道、天宪最浓，神性极强。
- 栖居者：先天神灵、高境守序者、古老星君系残脉。
- 风险：越往上越接近无情秩序。

这里不适合低阶生灵久留，因为规则太“硬”，一切有情与杂念都会被不断削磨。

### 2. 人间层

- 能量特征：灵气、承运、香火、劫波彼此纠缠。
- 栖居者：绝大多数凡人、修士、妖族、神道附属。
- 风险：最稳定，也最持续流血。

人间层是所有主线真正发生的地方，因为一切高处与深处的代价，最后都要压在这里。

### 3. 九幽之下

- 能量特征：秽气、绝煞、逆业、死煞最重。
- 栖居者：渊魔种、阴祟群、诡种核心、古老封印物。
- 风险：规则会被持续污染和扭曲。

九幽不是单纯的地下空间，而是一整套“秩序被不断磨穿”的深层世界。

## 四、重要地理隔断

扶摇界的大地之所以没有轻易被统一，是因为有几种天然或人为形成的地理隔断。

### 1. 断龙岭

- 本质：一条被上古大战斩断的超级地脉。
- 作用：隔绝中州与西陲的灵材流通。
- 风险：常年逸散绝煞，能吞法器、蚀血肉。

### 2. 归墟大泽

- 本质：多条古水脉塌陷后形成的超级泽国。
- 作用：隔断人间王朝与山海妖域的直接推进线。
- 风险：水府、沉城、旧神遗庙混杂，极易迷失。

### 3. 天风海

- 本质：高空风层与海渊潮汐交汇形成的异相海域。
- 作用：使正常飞渡与海渡都极其危险。
- 风险：常有空间偏移，容易误入折叠秘境。

### 4. 白骨原

- 本质：古王朝与神军决战后的死地。
- 作用：把两大文明板块彻底切开。
- 风险：阴祟、兵魂、怨力循环不息。

### 5. 镇渊长城

- 本质：不是普通城墙，而是一条横贯边荒的阵线型地理工程。
- 作用：隔断深渊回潮最直接的出口。
- 风险：一旦失守，不是丢一地，而是整片边疆一起烂掉。

## 五、洞天、福地、秘境与副本

扶摇界的“副本”不能做得像游戏房间，而要和世界伤口、历史遗留、空间折叠直接相关。

### 1. 洞天

- 定义：依附于稳定地脉或高位规则形成的折叠小天地。
- 特点：相对稳定，可长期经营。
- 常见用途：宗门山门、闭关地、药园、传承地。

### 2. 福地

- 定义：能量浓郁、规则温和、适合养人养物的特殊地域。
- 特点：未必折叠，但一定稀缺。
- 常见用途：立宗、养脉、培育天材地宝。

### 3. 秘境

- 定义：古代遗留的半封闭空间，往往带有明确时代烙印。
- 特点：周期性开启，不够稳定。
- 常见用途：试炼、争宝、揭露古史。

### 4. 裂隙副本

- 定义：规则裂痕短暂外翻后形成的危险区域。
- 特点：极不稳定，奖励和死亡率都极高。
- 常见用途：主角线的高风险奇遇、世界真相碎片投放点。

### 5. 诡域

- 定义：由怨力、怪谈和坏掉的规则共同维持的局部异化世界。
- 特点：进去容易，出来难；很多时候要先破解其逻辑才能离开。
- 常见用途：诡种、阴祟主场。

## 六、地图与故事推进的关系

主线地图最好遵循这条升级路线：

1. 边荒小城
2. 一州宗门
3. 王朝腹地
4. 山海妖域
5. 天下核心
6. 九天旧境
7. 九幽深层
8. 世界伤口最深处

这样地图升级会自然带动战力升级、视角升级和真相升级。

## 七、地理资源分布铁律

1. 地脉厚重处，宗门与王朝必争。
2. 水脉密集处，贸易、信息和黑市最发达。
3. 古战场附近，秘境和诡域最密集。
4. 边荒裂伤带，最容易诞生灾变种与逆天者。
5. 高空与深渊越极端，普通生命越难久居。

## 八、一句话总结

扶摇界的地图，不是一张平面图，而是一张由地脉、历史、污染、折叠空间与文明边界共同叠出来的活地图。哪里风景越壮阔，往往就藏着越古老的伤口。
