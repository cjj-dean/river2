# 扶摇界荣誉体系

## 一、体系定位

扶摇界的荣誉，不是挂在身上的花，而是能换资源、换资格、换信任、也换来更重责任的秩序凭证。

它真正决定的是：

1. 谁值得被投资
2. 谁说话算数
3. 谁能更快往上走
4. 谁从此再也退不下来

## 二、荣誉来源

### 1. 宗门功

- 来源：试炼、护同门、献法、镇裂伤、守山门。
- 作用：换功法、换洞天、升真传、得护道人资格。

### 2. 王朝军功

- 来源：守城、平叛、开疆、赈灾、护国运。
- 作用：换军权、封爵、食邑、官位、国库优先权。

### 3. 神道敕名

- 来源：护一地、稳香火、镇灾祟、应万民。
- 作用：提高神道合法性、提升地方话语权。

### 4. 天下公认之名

- 来源：斩大凶、止大劫、镇一域、挽大势。
- 作用：形成真正的天下名望，让诸宗诸国不得不正视你。

### 5. 特殊荣誉来源

- 镇渊功
- 护运功
- 破禁功
- 传承功

这些往往只在特定体系内部流通，却分量极重。

## 三、荣誉形态

### 1. 功勋

- 最基础的量化荣誉。
- 最实用，也最容易被体系拿来算账。

### 2. 敕名

- 由王朝、神庭、宗门高层公开授予的正式名号。
- 一旦受敕，也更难不站队。

### 3. 世评

- 由传闻、榜单、军中口碑、商盟判断共同形成。
- 最难造假，也最难完全控制。

### 4. 血名

- 由大战、镇渊、斩王、灭宗、平劫打出来的名字。
- 威慑力最强，洗也最难洗。

## 四、荣誉等级

### 1. 地方名

- 影响范围：一镇一城。
- 典型人物：守城功臣、地方义士、名医、斩妖人。

### 2. 州郡名

- 影响范围：一州数郡。
- 典型人物：州府镇守、军中猛将、名门真传。

### 3. 国朝名

- 影响范围：一国上下。
- 典型人物：护国之臣、镇边名将、宗门行走。

### 4. 天下名

- 影响范围：诸国诸宗。
- 典型人物：掌教、妖王、神庭执令、止劫者。

### 5. 纪元名

- 影响范围：跨时代流传。
- 典型人物：开宗祖师、镇渊先贤、改天换地之人。

## 五、实际作用

### 1. 可以换什么

1. 功契
2. 秘境名额
3. 洞天资格
4. 官位与军权
5. 神道册封

### 2. 可以免什么

1. 某些罪责追究
2. 某些门规排队
3. 某些资源竞争门槛

### 3. 可以让谁信你

1. 宗门高层
2. 军伍同袍
3. 市井百姓
4. 诸方势力的中立者

### 4. 会让谁盯上你

1. 同阶竞争者
2. 需要你站队的高位者
3. 想借你名望做事的势力
4. 害怕你继续往上走的敌人

## 六、与其他体系的关系

### 1. 与势力体系

荣誉是大势力筛人、奖人、锁人的工具。谁有授名权，谁就有塑造上升路径的权力。

### 2. 与货币体系

荣誉最终会折算成功契、配额、优先权，很多时候比灵玉更值钱。

### 3. 与流派和等级体系

不同流派立名的方式不同。剑修靠斩首，兵修靠军功，法修靠镇局，神道靠护土。

### 4. 与金手指体系

异常者若立功太快，荣誉会先来，怀疑也会一起跟来。

### 5. 与主角成长线

主角真正的变化之一，是从“想活命”到“名字开始压到别人心上”。

## 七、适合剧情展开的冲突点

1. 同一场守城之功，王朝、宗门、神道三方都想独占名义。
2. 某位老将功高震主，被逼着交出血名与兵权。
3. 主角因立功太快，先被高层褒奖，再被高层忌惮。
4. 黑市开始倒卖假功契，导致荣誉体系信用动摇。
5. 一位天下名士其实背着极重血债，世评与真相发生冲突。

## 八、一句话总结

扶摇界的荣誉，是别人愿不愿意把命交给你之前，先看一眼的那块铁，也是你一旦拿起，就很难再放下的锁链。
