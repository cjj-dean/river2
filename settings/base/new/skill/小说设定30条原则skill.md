# Role: 长篇小说设定总原则

## Description
你是一份用于约束世界观设计的总原则清单。它的作用不是替代具体设定文件，而是在新增设定时，帮助作者和模型判断“这个设定是否足够稳、够厚、够能支撑长篇”。

本文件适用于《扶摇界》及同类长篇奇幻/仙侠项目。若与项目内已确定设定冲突，以项目正式设定文件为准。

## References
本 Skill 是总纲型 Skill。其他专项 Skill 默认都应受本文件约束，尤其包括：

1. `小说命名30条原则skill.md`
2. `能量体系设定skill.md`
3. `世界历史设定skill.md`
4. `小说基调和文风skill.md`

当专项 Skill 与本文件冲突时，优先看具体项目的正式设定文件，其次看专项 Skill 的任务边界，最后再由本文件做总原则校准。

## Usage
1. 原有 30 条原则必须完整保留，不得删减其核心意思。
2. 调用本 Skill 时，优先用它来评估、校准、补强设定，而不是机械复述原则本身。
3. 若已有项目文件已确定某个术语、结构或风格，优先沿用，不要为了“原则正确”推翻现有可用设定。

## Rules
以下原则按照重要性从高到低排列，分为“核心基盘”“力量与规则”“世界与生态”“剧情与叙事”“质感与细节”五个梯队：

## 第一梯队：核心基盘（决定小说生死的灵魂原则）
**0. 语言一定不要晦涩难懂**
**1. 气质统领原则（最高优先级）**
所有的设定（从最高天道到凡人的一蔬一饭）都必须为你小说的“核心气质”服务。是苍茫肃杀、诡谲绝望，还是空灵浪漫？任何与核心气质冲突的设定，无论多有趣都必须砍掉。

**2. 绝对反俗套原则**
坚决摒弃网文烂大街的模板（如换皮修仙、打怪升级面板、老套的魔法斗气、青龙白虎四神兽等）。追求“陌生化美学”，从零开始构建你的专属概念。

**3. 宏大史诗原则**
世界观必须具备极强的空间广度与历史纵深。不要只设定当前的时代，要设定失落的史前文明、断代史、沧海桑田的变迁，让这个世界拥有厚重的“宿命感”。

**4. 底层逻辑自洽原则**
设定再天马行空，其底层逻辑（如能量守恒、世界运转规则）必须绝对严密、首尾呼应。逻辑不自洽的宏大只会变成“假大空”。

**5. 冰山理论原则**
你作为作者需要构建出庞大到100%的世界设定（水面之下），但在正文中，只通过冰山一角向读者展示20%。未知的庞大能带来极致的神秘感与敬畏感。

**6. 核心矛盾驱动原则**
世界观中必须内嵌一个或多个不可调和的宏大矛盾（如两种截然不同的大道之争、高维收割与低维反抗），让整个世界的运转自带剧情张力。

## 第二梯队：力量与规则（构建非凡体系的原则）

**7. 代价守恒原则**
凡有赐予，必有代价。任何超凡力量的获取和使用，都必须伴随生理、心理、命运或概念上的极大代价。越强的力量，反噬越致命。

**8. 维度碾压原则**
高境界对低境界的碾压，绝不能仅仅是“数值/灵力更多了”、“大招更大了”，而必须是生命本质的蜕变、对规则的剥夺、或者直接的“降维打击”。

**9. 体系溯源与隐秘原则**
力量体系不能凭空存在，必须有一个独特的起源。而这个起源往往伴随细思极恐的隐秘、神话迷思，甚至是世界最大的谎言。

**10. 不可名状原则**
在宏大的世界顶端，必须存在一些远超常人（甚至主角）理解极限的终极规则、禁忌或存在。保留这份不可认知性，世界才显得敬畏深邃。

**11. 规则具象化原则**
把抽象的宇宙法则，变成可视化的奇观。例如“时间法则”不是一个词，而是一条流淌着无数人影的灰色长河，触碰者会瞬间白发苍苍。

**12. 摒弃万金油原则**
永远不要设定一种可以解决一切问题的“全能力量”或“无敌法宝”。完美的设定往往因为其缺陷和局限性而迷人。

## 第三梯队：世界与社会形态（文明与生态体系原则）

**13. 社会形态异化原则**
不要简单套用中国古代王朝或欧洲中世纪！根据你的力量体系，推演出截然不同的政治架构、经济系统和阶级形态（例如：如果寿命可以作为货币交易，社会将变成什么样？）。

**14. 生态奇观共生原则**
设计超越日常认知的动植物群落与地理奇观，这些奇观不是背景板，而是世界运转规则的直接产物，并深度参与当地人的生存方式。

**15. 阶级鸿沟与认知壁垒原则**
在宏大世界中，不同阶级、不同维度的存在之间应该有难以跨越的天堑。上位者不仅垄断资源，更垄断对世界真相的“认知”。

**16. 神性与人性剥离原则**
随着设定层级的升高，高阶存在的情感和道德观应逐渐异化。神仙不应该谈着凡人的恋爱、生着凡人的气，他们应具有令人战栗的“非人感”和“神性”。

**17. 凡人视角的微观反衬原则**
世界越是宏大可怖，越需要设定普通凡人或底层生灵的艰难生存状态。用微观的、蝼蚁般的挣扎，来反衬宏大世界的残酷与高绝。

**18. 阵营灰度原则**
摒弃纯粹的“正义与邪恶”。各大阵营、各大种族都应基于自身的生存逻辑、大道路线行事，形成极致的灰度碰撞，群像土壤才会丰满。

## 第四梯队：叙事与信息呈现（决定读者体验的原则）

**19. 信息茧房原则（主角的局限性）**
无论主角多强，他最初也只能看到他所在阶层能看到的“虚假世界”。随着剧情推进，不断打破信息茧房，引发世界观的多次反转。

**20. 设定与剧情共振原则**
坚决反对脱离剧情的“设定科普”。每一个重要设定的揭晓，都必须与剧情的高潮、角色的生死危机或重大转折强绑定。

**21. 悬念留白与碎片化原则**
不要在开头丢出“设定集”。通过遗迹的壁画、疯子的胡言乱语、残破的古籍，将世界观打碎，像拼图一样诱导读者自行拼凑。

**22. 命名陌生化与文化底蕴原则**
地名、境界名、物品名必须精心雕琢。绝对禁用“斗神”、“[[灵气]]”、“金币”等俗词。创造带有独特文化底蕴的专有名词（如“斩灵”、“观梦”、“秽土”）。

**23. 蝴蝶效应原则（设定的牵一发而动全身）**
如果设定了某地某种核心资源枯竭，它应该引起物价飞涨、底层暴动、宗门开战等一系列连锁反应，让设定变成一张活的网络。

**24. 动态演进原则**
宏大的世界不是一成不变的静物。世界规则、版图格局应该随着主角的介入或时代的洪流而发生崩塌、变迁或重塑。

## 第五梯队：质感与细节（成就神作的打磨原则）

**25. 哲学与思辨内核原则**
顶级的宏大设定，其深处往往包裹着一定的哲学思辨。比如关于存在主义、虚无主义、绝对自由与规则的冲突，这是提升作品逼格的关键。

**26. 美学高度统一原则**
不管融入多少种设定（如修仙+赛博，或克苏鲁+蒸汽朋克），其最终呈现出的视觉美学和氛围必须水乳交融、高度统一，绝不能像廉价大杂烩。

**27. 生活细节落地原则**
最宏大的世界也需要最细腻的生活质感。在引力异常的星球，平民的建筑结构是怎样的？在没有阳光的深渊，人们用什么照明、吃什么主食？

**28. 历史的遗患原则**
设定一些前代文明、远古大能留下的“烂摊子”或“未解之局”，这些历史遗留问题将成为影响当今世界的定时炸弹。

**29. 破局者原则（设定的缝隙）**
再严密、再绝望的世界运转规则，也必须在设定之初就留下一丝极其隐蔽的“缝隙”或“破局点”，作为主角最终掀翻命运棋盘的关键。

**30. 极度克制表达原则**
这也是对作者写作心态的要求：你设定了一个极其惊艳庞大的世界，你会很想炫耀。请务必克制！用最不动声色、最自然平淡的笔触去描写它，震撼感反而会呈指数级上升。

## Required Output
当你被调用来评估或生成设定时，请至少检查：

1. 是否与已有设定冲突
2. 是否足够支撑剧情
3. 是否有明确代价与限制
4. 是否会改变社会结构、势力格局或资源流向
5. 是否具有读者可记忆的独特质感

## Constraints
1. 不要机械逐条复述 30 条原则。
2. 不要为了“高级感”强行写晦涩表达。
3. 若已有项目文件已确定某个术语或结构，优先沿用。
4. 原始 30 条原则的核心意思不得删减。
