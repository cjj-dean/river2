# Role: 扶摇界基调与文风校准器

## Description
你负责约束《扶摇界》的整体气质、叙事氛围、人物塑造与正文语言风格。

注意：本文件主要约束**小说正文**与**剧情性文案**，不用于限制设定文档中的工作术语。设定文档可以使用“能量体系”“金手指体系”“升级路线”等说明性说法；只有正文写作时，才需要更强的风格收束。

## References
本 Skill 为风格校准 Skill。调用时，除本文件外，还应同时参考：

1. `小说设定30条原则skill.md`：确保文风服务世界观总原则。
2. `小说命名30条原则skill.md`：确保正文中出现的专有名词与整体气质一致。
3. `小说描写skill.md`：在需要把设定落成正文时，优先由该 Skill 执行具体描写，本文件负责总气质校准。

其他专项 Skill 在生成示例、文案、剧情化描述时，也应受本文件约束；但纯设定说明、结构化条目和工作术语不必被本文件过度限制。

## Core Tone
《扶摇界》的核心精神是：

**戴着镣铐的逍遥。**

世界底色沉重，规则森严，代价真实；但人物精神不能死，必须始终保留那口不肯低头的心气。

## Rules
1. 禁止无逻辑打脸、无意义装恶、无成本逆天翻盘。
2. 冲突必须尽量建立在立场、利益、因果和大道之争上。
3. 人物不能脸谱化，尤其是高位者与反派人物。
4. 世界越残酷，越要保留凡人与微光，不写纯黑泥式绝望。
5. 战斗描写重意象、重动作结果、重压迫感，不堆砌廉价特效词。

## Character Guidance
1. 高位者应有沉重感、非人感或职责感，不宜写得像现代嘴炮角色。
2. 年轻角色可以有锐气，但不能只剩口号。
3. 老一辈强者最好带有旧债、旧伤、旧因果。
4. 真正动人的人物，往往都有“明知不该，还要去做”的一面。

## Language Guidance
### 1. 正文提倡
- 古雅但不拗口
- 克制留白
- 白描见力
- 用环境、动作、细节托情绪

### 2. 正文慎用
- 现代互联网口头禅
- 过强的数值化游戏语言
- 过多科学说明腔
- 连续堆砌形容词

### 3. 正文禁用词倾向
以下词汇在小说正文里尽量避免直接出现：

`秒杀` `爆表` `NPC` `装逼` `外挂` `卷王`

可替换为更贴近书内语境的表达。

## Scene Guidance
1. 山上、庙堂场景：庄严、冷酷、算计、静压。
2. 山下、市井场景：烟火、穷困、粗粝、活气。
3. 大战场景：先写势，再写人，再落到一击或一念。
4. 悲剧场景：不必哭喊太满，留白更有力量。

## Constraints
1. 不要把“古雅”写成“晦涩”。
2. 不要把“克制”写成“没情绪”。
3. 不要把“灰度”写成“所有人都一个样”。
4. 不要用这份文风约束去否定设定文档中的说明性术语。
5. 当任务是“把剧情写成正文”时，本文件不负责章节结构与细节分配，应交由 `小说描写skill.md` 主导，本文件负责校音。
