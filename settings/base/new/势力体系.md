# 扶摇界势力体系

## 一、总纲

扶摇界的势力，不是简单的“宗门、王朝、妖族、魔道”并列，而是围绕三件事展开：

1. 谁占着地脉节点。
2. 谁承着香火与承运。
3. 谁有能力替天下镇压天缺、浊潮与劫波。

因此，扶摇界任何一个真正的大势力，都必须同时具备至少两种能力：

1. **暴力能力**：能打。
2. **治理能力**：能养人、控地、守线。
3. **合法能力**：能让自己的统治被某种秩序承认。

少一样，只能算强者聚集地，算不上真正的大势力。

## 二、天下六大势力母类

### 1. 山上宗门

- 核心资源：功法、传承、洞天、地脉、护山大阵。
- 核心职能：修士培养、裂伤镇压、对外征伐。
- 统治方式：师承、门规、资源分级、护道人体系。
- 优势：战力高度集中，高手产出最稳定。
- 缺陷：容易脱离凡俗，视众生为成本。

宗门不是纯学术组织，也不是单纯武力团体，而是“掌握超凡生产力”的山上政权。

### 2. 人间王朝

- 核心资源：人口、粮道、税赋、军伍、城池、国运。
- 核心职能：整合凡俗社会、承接劫波、维持大范围秩序。
- 统治方式：郡县、军镇、祭天体系、香火官祀。
- 优势：组织能力最强，能把散乱人口变成可运转的国体。
- 缺陷：越庞大越离不开压榨，承运一断便是大崩。

王朝不是宗门附庸，也不是纯粹背景板。很多时候，山上负责“镇局部”，王朝负责“扛全局”。

### 3. 神道体系

- 核心资源：香火、庙宇、信众认知、地方祭礼。
- 核心职能：镇灾、护土、调和一地秩序与民心。
- 统治方式：神位册封、庙系分层、祭祀网络。
- 优势：对局部灾变反应最快，最容易直接影响一地百姓。
- 缺陷：最容易被认知绑死，最后失去自我。

神道是王朝与天地之间的缓冲层，也是最容易变成怪物的秩序体系。

### 4. 妖域势力

- 核心资源：山海古域、水府龙脉、血脉祖地、古老天象。
- 核心职能：维护异族生存空间、争夺合法地位、保留血脉传承。
- 统治方式：血脉、战功、祖训、古盟。
- 优势：单体生命层次高，地域压制力极强。
- 缺陷：内部极难统一，派系差异比人族更大。

妖域势力不是一个整体，而是多个互相竞争的“大泽水府”“山海祖庭”“边荒妖国”拼出来的总称。

### 5. 逆天者组织

- 核心资源：真相、禁术、旧时代遗产、秘密网络。
- 核心职能：揭露旧秩序、破坏镇压节点、寻找替代方案。
- 统治方式：松散盟约、理念认同、任务链条。
- 优势：理念锋利，最接近世界真相。
- 缺陷：容易因为目标太大而忽视局部代价。

逆天者不是一个“邪教”，而是一切不愿继续替天还债之人的总称。其中有疯子，也有真正清醒的人。

### 6. 浊渊群落

- 核心资源：秽气、绝煞、劫波、规则裂缝、古老污染源。
- 核心职能：不是治理，而是侵蚀、扩张、撕裂旧秩序。
- 统治方式：吞噬、寄生、共鸣、污染传播。
- 优势：最难用正常政治与战争手段处理。
- 缺陷：很难形成稳定文明。

浊渊群落不算传统意义上的国家或宗门，但在故事功能上，它们就是世界失衡后的“第六大势力面”。

## 三、势力层级划分

扶摇界的势力强弱，不能只按人数或高手数量定，而要按其能影响多大范围的天地结构来分。

### 1. 末流势力

- 典型形态：寨子、小帮、山门支脉、小城香火庙。
- 控制范围：一镇一山一河。
- 特点：靠一个高手或一块小地盘吃饭。

### 2. 地方强权

- 典型形态：州郡宗门、军镇世家、中型水府。
- 控制范围：一州数郡或一片山泽。
- 特点：已经能影响地方税赋、灵材流通、军力调配。

### 3. 一方霸主

- 典型形态：顶尖宗门、封疆王府、妖域祖庭。
- 控制范围：数州之地或一整片古域。
- 特点：能独立镇压一处大裂伤，能改写地方格局。

### 4. 天下级势力

- 典型形态：国朝、中州大宗、神庭总系、跨域大盟。
- 控制范围：影响整个扶摇界一部分基础秩序。
- 特点：不仅能争资源，还能定义“什么算合法”。

### 5. 纪元级存在

- 典型形态：太古神系残脉、上古镇渊联盟遗产、九天旧主布局。
- 控制范围：跨时代。
- 特点：它们未必还公开存在，但其遗产仍在决定今天天下怎么活。

## 四、扶摇界势力运转铁律

1. **没有地脉，就没有长期立足之本。**
2. **没有人口与供养，再强的修行势力也会空心化。**
3. **没有合法性，再大的胜利也只是短期占山。**
4. **镇压天缺的能力，决定一个势力是否配得上天下地位。**
5. **承运不是奖励，而是更重的债。**

## 五、四种典型权力结构

### 1. 宗门型权力

特点是“师承高于血缘，资源高于土地”。

这类势力看重功法、师门、悟性、战力排序，适合写天骄、派系斗争和传承之争。

### 2. 王朝型权力

特点是“土地高于个人，整体高于局部”。

这类势力擅长调兵、征税、赈灾、祭天，适合写现实、秩序、冷酷大局。

### 3. 神道型权力

特点是“认知即统治”。

谁能被更多人相信，谁就能更稳地存在。它天然适合写信仰、异化、认知战争。

### 4. 妖域型权力

特点是“血脉与祖地先于法统”。

它更像是把地理、血统、古盟和猎场拼在一起的文明结构，适合写异族政治与古老盟约。

## 六、适合主线的核心势力格局

为方便故事展开，扶摇界现世格局建议设为：

1. **中州大宗联盟**：名义上镇守天下，实则内部裂痕极深。
2. **大离王朝**：当前最强人间国体，承运正盛但已显疲态。
3. **社稷神庭**：遍布各城各州的官祀神系，与王朝深度绑定。
4. **山海妖盟**：松散妖族联盟，内部有主和主战两派。
5. **逆命同盟**：逆天者最大的地下网络。
6. **九幽渊巢**：浊渊回潮最活跃的污染源之一。

## 七、势力之间最稳定的冲突源

1. 宗门与王朝争谁有资格定义天下秩序。
2. 神道与宗门争香火、地脉和地方主导权。
3. 王朝与妖域争边疆和合法国祚。
4. 逆天者与一切守序势力争“这座世界到底值不值得继续这样运转”。
5. 浊渊序列逼所有势力不得不合作，却又让所有势力更容易互相甩锅。

## 八、一句话总结

扶摇界所谓势力，不过是一群有能力替天下扛债、也有能力让别人替自己扛债的人，围绕地脉、人口、香火与真相，搭起来的巨大权力结构。
