# 第一卷 边荒活命

## 第07章 旧渠岔口

### 章节信息

- 当前版本总字数：2926
- 章节内容：众人在旧渠岔口围绕许沉舟旧痕与路径选择展开争论；左渠像在引人，右渠更死却不迎人；许安生第一次更明确地依靠自身感知帮众人挑路，并听见黑里有人叫他的名字。
- 新出场人物：无
- 新出现的地理：右侧旧路、左侧水路、旧渠更深层通道
- 新出现的势力：无新的明面势力，但许沉舟旧线和旧渠旧布局进一步坐实

众人都没动。

旧渠里的水声本就细，这一刻却像被谁用手按住了，只剩黑暗深处那一下极轻的回响，还在石壁之间慢慢荡。

像是有什么东西，隔着看不见的水和年头，朝他们望了一眼。

赵二狗先绷不住，声音压得极低，还是发颤：“这下头……真有活物？”

谢停山没理他，只盯着右侧那面石壁。

壁上的“沉舟”二字太淡，若不是许安生眼尖，换了谁来都未必认得出来。可越是这样，越显得那两笔字瘆人。像是一个本该烂在旧事里的名字，隔了许多年，忽然又从墙缝里往外渗了一点。

许母的手紧了又松，松了又紧，最终还是抬起手，在那两个字上轻轻碰了一下。

石壁潮湿，冰凉，手指一落上去，立刻沾了层细水。

“是他的字。”她轻声道。

许安生转头看向她。

许母极少用这种笃定口气说起许沉舟。平日里别人问，她不是不说，就是一句“人早没了”打发过去。今晚却不同。黑山城已经烂了，城里的人和规矩都在往下塌，许多原本死死压住的话，竟也像压不住了。

陆川河站在后头，脸上的那点镇定终于也裂了一道缝。

“他来过这里？”许安生问。

“不止来过。”谢停山声音发沉，“这条渠，最早就是他找到的。”

赵二狗怔住：“你爹还会钻下水道？”

这话一出口，连他自己都觉得不对，忙缩了缩脖子，往后退半步。许安生却没顾上理他，只盯着谢停山。

“那为什么从来没人提？”

“提这个做什么？”谢停山瞥了他一眼，“提了，黑山城就能不死人？”

许安生被这句噎了一下，喉咙发紧。

谢停山说得不算错。黑山城这种地方，活人每天都在被别的事推着走。矿税、徭役、军粮、药价、冻死的人、饿死的人，桩桩件件都足够压得人直不起腰。旧年里死了个边军小校，没留下尸首，也没留下功名，谁又会真的记得。

可他还是觉得心口堵得厉害。

那毕竟是他爹。

他从没见过的人，如今却像一截埋在地底的朽木，越往下刨，越露出根系来。

“这边不能走。”陆川河忽然开口。

众人一起看向他。

陆川河抬了抬下巴，指着刻字那条窄渠：“右边是旧路。许沉舟能走，不代表现在还能走。那时候这地方没烂成这样。”

“你怎么知道？”谢停山问。

“因为我走过。”陆川河答得很平，“走进去，活着退出来的，就我一个。”

赵二狗的脸一下更白了。

许安生却听出了别的东西：“什么时候？”

“你爹死后第三年。”陆川河看着那面墙，像在看一口旧井，“那时右渠里已经开始有东西叫人名字了。”

旧渠里本来就冷，这句话一落，连风都像更阴了一分。

“叫人名字？”赵二狗嘴唇发干。

他从小胆子不算太小，敢偷矿棚里的酒，也敢翻城根底下的鸟窝，可那些胆子都只是对活人和寻常东西。真一听见“有东西会在黑里叫你名字”，那点装出来的横立刻就碎得差不多了。

他从小胆子不算太小，敢偷矿棚里的酒，也敢翻城根底下的鸟窝，可那些胆子都只是对活人和寻常东西。真一听见“有东西会在黑里叫你名字”，那点装出来的横立刻就碎得差不多了。

“嗯。”陆川河淡淡道，“先叫你乳名，后叫你爹娘，再往后，叫你自己都不肯认的旧事。你若回头，便得留在里头陪它。”

赵二狗听得腿都软了：“那……那还站着做什么，走左边啊！”

谢停山没立刻动。

他看着岔口左边那道更宽的水路，眼神一点点沉下去。

“左边有风，说明能通。”他说，“可风太稳了。”

“稳还不好？”赵二狗几乎要哭出来。

“在这种地方，太稳才不对。”许安生忽然接了一句。

众人都看向他。

许安生自己也愣了一下。他原本只是顺着心里那股说不清的别扭，把话吐了出来。可说出来之后，反倒越想越明白。

这渠底一路过来，石缝漏水，暗苔滑脚，哪儿都带着死气和旧年的烂味。偏偏左边那条路，风顺得过分，像有人在尽头拿把扇子，不紧不慢地朝这边送。

像路。

也像引。

“你看见什么了？”许母立刻问。

许安生闭了闭眼，握紧黑刀，再往左边那条水路望去。

他眼前那层薄灰又浮上来了。

渠底的黑暗一下分出深浅。左边那条路表面平，底下却像压着一层细细的黑丝，一缕一缕顺着风往这边摇，像人在水里散开的头发。

而右边那条窄渠，虽然暗得更深，却更“死”。

不动，也不迎。

许安生慢慢吐出一口气：“左边不干净。”

这句话一出口，连他自己都觉得喉头发涩。他其实也说不清所谓“不干净”到底是什么，只知道那风不该那样稳，那水不该那样平，那股往这边来的劲，更像一只手，正隔着黑暗不紧不慢地招人。

这句话一出口，连他自己都觉得喉头发涩。他其实也说不清所谓“不干净”到底是什么，只知道那风不该那样稳，那水不该那样平，那股往这边来的劲，更像一只手，正隔着黑暗不紧不慢地招人。

谢停山盯着他：“右边呢？”

“右边也不是好路。”许安生低声道，“但它至少没在等我们。”

这话说得赵二狗头皮发炸，嘴唇动了两下，终究没敢插嘴。

陆川河沉默片刻，忽然笑了一下。那笑意很浅，也很冷：“你爹当年也是这么挑路的。”

许安生心里一紧。

“然后呢？”

“然后他把活路让给了别人。”陆川河道，“自己死在了后头。”

这一句像石头砸进水里，砸得许安生胸口发闷。

许母脸色发白，却到底没再追问。眼下不是问旧账的时候，问得再多，人也回不来。最要紧的，是把眼前这口气先保住。

谢停山终于拔步，走向右边窄渠。

“我开路。”他说。

“我殿后。”陆川河提刀跟上。

赵二狗赶紧往中间缩：“我、我跟着许安生。”

“你最好闭嘴跟着。”谢停山头也不回，“再胡乱出声，我先把你嘴堵了。”

赵二狗立刻闭上嘴。

右渠比方才那段还窄，顶低得很，高些的人都得稍稍低头。石壁两侧全是青黑水痕，脚下的水没过靴底，踩下去冰得骨头都发酸。往前不过十几步，后头岔口的那点光便彻底没了。

这一下，众人才真正像是钻进了黑山城的肚子里。

许安生走在中间，手始终压着腰侧黑刀。刀是冷的，他掌心却在出汗。四下太静，静得只能听见众人的呼吸、水声，以及很远很远的地方，似有若无的一点回响。

起初他以为是自己听错了。

可走出一段后，那声音又来了。

极轻，像有人隔着厚墙，低低喊了一声。

“……安生。”

许安生后背猛地绷直。

声音是从后头传来的。

不是许母的声，也不是赵二狗的声。更老一点，更低一点，像是从许多年以前的风里吹过来的。

他脚步几乎要停。

下一刻，谢停山的声音在前头冷冷响起：“别回头。”

这三个字落下，像一桶冰水从头浇下。

许安生猛地咬住舌尖，血腥气一下冲散了耳边那点恍惚。再抬脚时，他才发现自己后背已经全湿了。

右渠果然也不是什么善地。

只是不像左边那样明着勾人，而是藏得更深、更旧，像专挑人心里最软那一点下手。

赵二狗显然也听见了什么，脸色难看得像纸，却死死咬着牙不敢出声，只一只手拽着许安生衣角，攥得发抖。

不知走了多久，前头的谢停山忽然抬手。

众人一下停住。

前方黑暗里，出现了一点极淡的冷光。

不是灯，也不是月。

像是水里漂着一层发白的皮。

陆川河在最后头轻轻吸了口气，嗓音压得极低：“坏了。”

“什么东西？”许母问。

谢停山没回头，只把刀慢慢提了起来：“沉尸皮。”

许安生还没明白这三个字的意思，那点冷光便动了。

像一张被水泡了很多年的脸，缓缓从渠底浮了上来。
