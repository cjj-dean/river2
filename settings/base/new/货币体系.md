# 扶摇界货币体系

## 一、体系定位

扶摇界的货币，不是单纯的金属交换物，而是把粮食、灵气、资格、香火与命运债，换算成可流通凭证的秩序工具。

它的真正作用有四个：

1. 维持凡俗生活。
2. 支撑修行资源流通。
3. 把资格与功劳折算成上升通道。
4. 让势力能够跨地域调度资源与人命。

## 二、货币种类

### 1. 基础货币：白铢

- 使用者：凡人、脚行、小商、农户。
- 来源：王朝官铸。
- 作用：日常衣食住行、小额税赋、雇工。

### 2. 大额货币：赤泉

- 使用者：州府、军镇、大商队。
- 来源：州库与国朝钱监。
- 作用：军需、粮道、车马、跨城大宗买卖。

### 3. 修行货币：灵券、灵玉

- **灵券**：由宗门、商盟、国库联合背书的资源票据，适合跨区域交易。
- **灵玉**：修行界硬通货，既能交易，也能直接修炼、布阵、养器。

### 4. 资格货币：功契

- 来源：宗门功勋、军功、护国功、镇渊功、神道敕记。
- 作用：换功法、换洞天名额、换官位、换庇护、换赦免。

### 5. 特殊货币：愿钱

- 来源：香火凝结与庙系祭炼。
- 作用：祭神、请符、换神道护佑、走灰色交易。
- 风险：最容易受信仰崩塌影响，贬值时甚至反噬持有者。

## 三、来源与信用逻辑

### 1. 谁铸造

1. 王朝掌白铢、赤泉。
2. 宗门、商盟、国库共同维护灵券信用。
3. 灵玉来自灵矿、洞天、地脉析晶，不是纯人造。
4. 功契由不同权力体系授予。
5. 愿钱由神道祭礼与庙系炼制。

### 2. 谁背书

1. 白铢、赤泉背后是税法、军队、粮仓。
2. 灵券背后是大势力间的资源信用。
3. 灵玉背后是其本身可直接转化为修行收益。
4. 功契背后是制度承认。

### 3. 为什么别人愿意认

因为在扶摇界，真正能流通的东西必须同时满足两点：

1. 能兑现。
2. 有人兜底。

没有势力承认的财富，只能算财货，算不上货币。

## 四、购买力分层

### 1. 凡人层

1. 几枚白铢：一顿粗食或一夜落脚。
2. 数十白铢：普通人家数日到半月开支。
3. 一枚赤泉：足够做一笔对凡人来说不小的生意。

### 2. 低阶修士层

1. 少量灵券：符纸、伤药、基础阵材。
2. 一块灵玉：短期修炼资源，或一件基础法器的部分成本。

### 3. 中高阶修士层

1. 数十到数百灵玉：丹药、护身法器、短期洞天资格。
2. 功契：往往比同值灵玉更难得，因为它直接决定能不能再上一步。

### 4. 大势力层

真正的大势力结算，很多时候早已不只是灵玉，而是：

1. 地脉节点
2. 洞天配额
3. 军镇粮道
4. 镇渊职责
5. 功契授予权

## 五、交易系统

### 1. 民间集市

- 货币：白铢、赤泉。
- 主体：百姓、小商、手工业者、脚夫。

### 2. 官市

- 货币：赤泉、部分灵券。
- 主体：州府、粮商、盐铁商、军需商。

### 3. 宗门坊市

- 货币：灵券、灵玉、功契。
- 主体：修士、炼丹师、炼器师、宗门附庸商会。

### 4. 跨域商盟

- 货币：灵券、赤泉、灵玉混用。
- 主体：大商会、水陆漕运、水府、边贸商队。

### 5. 黑市 / 鬼市

- 货币：灵玉、功契、愿钱，以物易物，甚至寿元和秘密。
- 主体：亡命徒、禁术师、掘墓客、灰线神道、异族中介。

## 六、与其他体系的关系

### 1. 与势力体系

谁掌哪套货币，谁就掌哪一层社会。

### 2. 与地理体系

地脉、商路、水脉、边关、秘境产出决定财富流向，绝地与裂隙则决定黑市兴衰。

### 3. 与流派和等级体系

不同流派的资源消耗不同，直接决定他们在交易系统中的位置。

### 4. 与金手指体系

最强的金手指不一定直接给钱，但一定能拿到“钱买不到的资格”。

### 5. 与荣誉体系

功勋和名望最终会折成功契、配额、敕名与优先权，这才是真正的上升路径。

## 七、适合剧情展开的冲突点

1. 一处灵矿枯竭，引发坊市暴涨、底层暴动、宗门开战。
2. 大离王朝赤泉贬值，州郡开始以粮代税，边军哗变。
3. 某宗门私铸灵券，导致跨域信用崩塌。
4. 黑市出现能买卖功契名额的灰线，引发天骄与废材身份倒挂。
5. 愿钱在一地失控反噬，庙神坠邪，整城交易体系崩盘。

## 八、一句话总结

扶摇界的货币，是把资源、秩序、资格和命运债，一层层折算成可流通凭证的巨大清算系统。
