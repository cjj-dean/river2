# [[扶摇界]]能量与法则体系

## 一、 能量的种类（清浊与正负的二元对立）
天人两极律：在[[扶摇界]]中，世界的能量并非单一存在，而是严格划分为“外在（宏观宇宙：大天地）”与“自身（微观宇宙：小天地）”两个维度。同时，万物皆具有两面性，能量在两个维度中均呈现出“正面力量”与“负面力量”的绝对对立与伴生关系。超凡者本质上是通过修炼自身微观宇宙的小天地，去撬动、抵抗或顺应宏观宇宙的大天地能量。

### 1. 外在力量（宏观宇宙：大天地）
| 力量层面    | 正面力量(秩序与生机)       | 负面力量(无序、狂暴、毁灭)            |
| :------ | :---------------- | :------------------------ |
| **规则层** | 大道                | 混沌                        |
| **能量层** | 天地灵气              | 天地秽气(魔气)                  |
| **大地层** | 地脉（大地之力）          | 绝煞                        |
| **因果层** | 天宪（浩然正气/功德/天道奖励）  | 逆业（业障、因果/天道惩罚）            |
| **信仰层** | 香火愿力              | 众生怨力（淫祠邪神、厉鬼的温床）          |
| **气运层** | 承运 （气运 / 国运 / 武运） | 劫波（亡国怨气 / 绝地凶煞 / 劫运 / 诅咒） |

### 2. 自身力量（微观宇宙：小天地）
| 力量层面    | 正面力量(求真与升维)       | 负面力量(沉沦与堕落)   |
| :------ | :---------------- | :------------ |
| **肉体层** | 命炉（气血 / 生机）       | 寂灭之力          |
| **精神层** | 镇心（意志之力/定力 / 剑意）  | 妄相（心魔/执念）     |
| **本性层** | **人性**（有情）        | **神性**（无情）    |
| **认知层** | 掌理（本命字 / 学问 / 道心） | 伪道（知见障）       |
| **情绪层** | 扶摇 （意气提升）         | 心陨（剑心破碎/心气衰败） |
| **命运层** | **宿慧**（前世遗泽）      | 命轨之缚（宿命枷锁）    |

- **【规则层】**
    
    - **正面：大道
        
        - **深度解释**：世界存在的绝对秩序与逻辑基盘。它不是某种具象的能量，而是“水往低处流”、“生必有死”这种不可忤逆的规则。领悟大道能掌握修改世界代码的权柄，但代价是会被这股绝对理性的力量同化，逐渐丧失个人的情感。
            
    - **负面：混沌
        
        - **深度解释**：大道的对立面，一切秩序的对立面。它既是世界的起点，也可能世界的最终的归宿。它表现为物质的崩溃（万象无定）、规则的变异（例如水往高处流、火焰冻结万物）
            
- **【能量层】**
    
    - **正面：天地灵气
        
        - **深度解释**：天地吐纳的至纯灵气，是万物演化和超凡者跃迁的基础能量。
            
    - **负面：天地秽气**
        
        - **深度解释**：超凡者吞吐源炁、大天地新陈代谢后排出的沉淀废料。它如无孔不入的毒瘴，凡人触之肉体畸变，修士吸入则会污染力量根基。世界底层的“深渊”就是天地秽气最聚集之地。
            
- **【大地层】**
    
    - **正面：地脉（大地之力）**
        
        - **深度解释**：星辰内部的能量神经网络。凡建都、立派，必寻地脉节点（龙脉）。借用地脉可以获得源源不绝的耐力与厚重如山的防御，但过度抽取会导致一方水土枯竭，化作荒漠。**地脉的强弱直接决定了该地域统治势力的承运高低。**
            
    - **负面：绝煞**
        
        - **深度解释**：地脉被强行斩断、或大地承受了超载的屠杀与灾变后，伤口处流出的“黑血”。绝煞之地寸草不生，能侵蚀一切法宝与肉体，往往孕育出没有任何理智的纯粹怪物。
            
- **【因果层】**
    
    - **正面：天宪（浩然正气/天道奖励）**
        
        - **深度解释**：顺应“太一”秩序、修补世界漏洞所获得的大天地认证。拥有天宪加身者，等于拥有了天道的“特权”，可以言出法随、万邪不侵，甚至走在路上都能捡到天材地宝。天宪加身者，天生免疫一切妄相；
            
    - **负面：逆业（业障/天道惩罚）**
        
        - **深度解释**：超凡者每一次逆天改命、杀戮掠夺，都会在大天地的账本上留下“欠条”。逆业一旦积累到临界点，大天地就会降下“天劫”或“人劫”进行强制清算。而身背逆业者，极易触发命轨之缚。
            
- **【信仰层】**
    
    - **正面：香火愿力**
        
        - **深度解释**：众生集体潜意识的结晶。能将凡人强行推上神坛，获得不可思议的神通。但这是一种“剧毒的蜜糖”，神明必须时刻回应信徒的期待，最终会被众生的认知裹挟，变成一个失去自我的泥塑木雕。
            
    - **负面：众生怨力**
        
        - **深度解释**：信仰崩塌、或者大规模屠杀后遗留的极度绝望与怨恨。它是大天地间最难清除的污染，会滋生出诡异的规则类怪谈、淫祠邪神，一旦爆发足以让一国之地化为鬼域。
            
- **【气运层】**
    
    - **正面：承运（气运/国运）**
        
        - **深度解释**：天地对某个势力或个人的“投资期”。处于承运状态时，人才井喷、逢凶化吉，仿佛整个世界都在为你让路。但气运如潮水，有涨必有退，盛极必衰是铁律。
            
    - **负面：劫波（亡国怨气/劫运/诅咒）**
        
        - **深度解释**：气运退潮后的反噬。当一个王朝覆灭、一个大教崩塌，曾经多强的承运就会转化为多可怕的劫波。它表现为连绵的灾荒、天骄的接连横死，是如同诅咒般的因果纠缠。

### 2. 自身力量（微观宇宙：小天地）

小天地是生命个体的熔炉，所有的力量都来自于对自我的挖掘、燃烧与对抗。

- **【肉体层】**
    
    - **正面：命炉（气血/生机）**
        
        - **深度解释**：肉体即是一座燃烧的熔炉，气血是其中的薪柴。超凡者通过点燃命炉来爆发出远超常理的破坏力。命炉越旺，肉体自愈极强、生机蓬勃，能以纯粹的血气冲散阴邪和**妄相（心魔）**。
            
    - **负面：寂灭之力**
        
        - **深度解释**：命炉过度透支后产生的灰烬。当修士频繁使用禁术、或寿命将尽时，体内会滋生寂灭之力。表现为肉体不可逆的枯萎、长出尸斑、甚至器官衰竭。这是一切生命最终走向死亡的倒计时。
            
- **【精神层】**
    
    - **正面：镇心（意志之力/定力/剑意）**
        
        - **深度解释**：灵魂深处的锚点，是劈开大天地威压的锋刃。它是不屈的信念被极度压缩后产生的实质化力量（如剑修的本命剑意）。镇心者，泰山崩于前而色不变，万般幻象不可加身。
            
    - **负面：妄相（心魔/执念）**
        
        - **深度解释**：镇心的伴生阴影。意志越强，光芒背后的阴影越深。它源于修士心底未解的遗憾、极度的恐惧或畸形的渴望。一旦镇心动摇，妄相就会反客为主，导致精神分裂或走火入魔。
            
- **【本性层】**
    
    - **正面：人性（有情）**
        
        - **深度解释**：凡尘的锚。爱恨情仇、悲欢离合。人性是修士力量的最初来源，也是在残酷修行界中保持“自我”不迷失的唯一凭证。
            
    - **负面：神性（无情）**
        
        - **深度解释**：生命向更高维度跃迁的必然代价。随着修为接近“太一”，修士会开始视万物为刍狗，失去共情能力，逐渐变成一个冰冷、完美的规则执行机器。神性越重，越强大，但也越接近“死亡（成为天道的一部分）”。
            
- **【认知层】**
    
    - **正面：掌理（本命字/学问/道心）**
        
        - **深度解释**：通过学习、顿悟，在自身小天地内建立起一套绝对自洽的微观法则（如儒家的本命字、道家的道心）。当你坚信你的“理”时，你就能在一定范围内用个人的认知去覆盖甚至扭曲外界的大天地。
            
    - **负面：伪道（知见障）**
        
        - **深度解释**：认知的崩塌与偏执。当修士发现自己坚信一生的“真理”其实是错的，或者被更高维度的逻辑击碎时，道心就会化为伪道。这会导致一身修为失控，轻则沦为凡人，重则变成狂信徒般的疯子。
            
- **【情绪层】**
    
    - **正面：扶摇（意气提升）**
        
        - **深度解释**：一种极其玄妙的“主观势能”。大胜之后、或者明悟本心时的极度自信，能让修士超常发挥，甚至越阶杀敌，犹如大鹏一日同风起，扶摇直上。
            
    - **负面：心陨（剑心破碎/心气衰败）**
        
        - **深度解释**：意气受挫后的精神崩塌。一次极其惨痛的失败、或者被彻底碾压自尊后，修士的心气会如同被戳破的皮球。哪怕肉体完好无损，也再也无法发挥出一半的实力，终生止步不前。
            
- **【命运层】**
    
    - **正面：宿慧（前世遗泽）**
        
        - **深度解释**：灵魂深处自带的“隐藏资产”。可能是前世大能转生的记忆碎片，或是远古血脉的突然觉醒。它能让修士生而知之，修炼一日千里，是打破阶层壁垒的超级外挂。
            
    - **负面：命轨之缚（宿命枷锁）**
        
        - **深度解释**：所有命运馈赠的礼物，都暗中标好了价格。宿慧带来多大的便利，命轨之缚就布下多深的陷阱。它是一张早已写好的剧本，注定了你要在某时某刻为某事付出代价，哪怕修为通天，也难以跳出这冥冥中的定数。


### 2. 不同能量类型之间的关系

宏观与微观、正面与负面能量之间存在着严密的克制、转化与代价关系：

- **大天地的正负对立**：
    
    - 代表秩序与大道的“规则”时刻在抵御无序和无常的“混沌”。
        
    - 修仙者赖以生存的基础“天地灵气”与腐蚀万物的“天地秽气”互为表里。
        
    - 顺应天道行事可获“浩然正气/功德”作为奖励，而违背常理则会受到天道惩罚，积累“业障和因果”。
        
    - 收集众生愿力可成“香火之力”，但一旦信仰崩塌，便会反噬为致命的“众生怨力”。
        
    - 势力的兴衰表现为气运、国运、武运等“好运”，与亡国怨气、绝地凶煞、劫运等“坏运”的博弈。
        
- **小天地的内化抗争（代价与限制）**：
    
    - **肉体层面**：气血与生机（肉体之力）是生命的根基，过度透支会引来寂灭之力、衰朽之气或死煞。
        
    - **精神层面**：强大的定力与剑意（意志之力）时刻面临心魔与执念的侵蚀。本命字、学问和道心（认知）如果出现偏差，就会沦为知见障、伪道或偏执。自信的“意气”一旦受挫，便会跌落为心气衰败、剑心破碎的“气馁”。
        
    - **灵魂/宿命层面**：有情众生的“人性”与无情高阶的“神性”是两条互斥的进化道路。即便拥有前世遗泽或绝顶天资等“宿慧”，也必须挣脱命劫、胎中迷或命中注定之死的“宿命枷锁”。
        

### 3. 能量的分布规律

- **整体分布特征**：能量在世界上的分布是绝对不均匀的，这种不均直接决定了天下势力的格局。能量聚集的洞天福地，必然被高等势力或强大物种垄断。
    
- **水平分布（地理疆域）**：
    
    - 在广袤的平原和中原腹地，往往汇聚着磅礴的“大地之力（地脉）”以及鼎盛的“国运”，这里是世俗皇权和顶尖宗门林立的中心。
        
    - 而在边疆或古战场遗迹等偏远地带，常年盘踞着“亡国怨气”与“诅咒”，形成无人敢轻易涉足的绝地。
        
- **垂直分布（高低位面）**：
    
    - **天空/九天之上**：越往高处，大天地的“规则（秩序、大道）”和“浩然正气”越发浓郁，此地多被追求“神性（无情）”的绝代大能或神明占据。
        
    - **深渊/九幽之下**：深渊底部是世界排泄废料的尽头，沉积着最纯粹的“混沌（无序）”、“天地秽气”与“业障”，孕育着以“寂灭之力”和“死煞”为食的恐怖异端物种。
