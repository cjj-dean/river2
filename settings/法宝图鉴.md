# 扶摇界法宝与物品图鉴

> **【文件说明与 Agent 校验指南】**
> 本文件是《扶摇界》小说的**法宝与物品体系详解**。
> **包含内容**：法宝分类、著名法宝、炼器炼丹、天材地宝。
> **Agent 约束准则**：
> 1. **品阶匹配校验**：法宝品阶必须与使用者境界匹配。下三境修士无法驾驭灵阶以上法宝。
> 2. **[[器灵]]设定**：只有灵阶以上法宝才可能诞生[[器灵]]，道阶法宝必定有[[器灵]]。
> 3. **法宝损毁**：法宝可以被损毁，损毁后需要大量资源修复。严禁"永不损坏"的无敌法宝。
> 4. **炼器材料**：炼制法宝需要合理的材料和时间，严禁"凭空炼制"。
> ---

## 法宝品阶说明

与功法一样，法宝分为四大阶：
- **凡阶**：凡俗铁匠打造，无灵气流转
- **法阶**：引灵气入物，分下、中、上品
- **灵阶**：去浊存清，可能诞生[[器灵]]，分下、中、上、极品
- **道阶**：融合[[大道残片]]，必有[[器灵]]，分下、中、上品

## 第一部分：著名法宝

### 一、道阶法宝（镇宗至宝）

#### 【[[太一拂尘]]】
- **品阶**：道阶下品
- **持有者**：[[赵黄庭]]（[[老天师]]）
- **外观**：通体雪白的拂尘，尘丝如银，根根分明
- **核心能力**：抹除因果
  - 可以抹除一个人在天道中的"存在痕迹"
  - 被抹除者会被世人遗忘，仿佛从未存在
- **来源**：融合了"因果"属性的[[大道残片]]
- **限制**：每次使用都会消耗[[老天师]]的寿元
- **[[器灵]]**：无名（一缕残缺的天道意志）

#### 【[[天衡残片]]】
- **品阶**：道阶核心残片（超越普通道阶）
- **持有者**：[[宁观]]（主角）
- **外观**：融于眉心识海，平时不可见
- **核心能力**：[[破妄之眼]]
  - 洞微：看破气机和破绽
  - 斩络：切断灵气连接
  - 窥命：观测因果和[[业障]]
- **来源**：[[天衡]]的核心残片，天道本源
- **限制**：使用过度会导致神识枯竭、双目流血
- **[[器灵]]**：无（或者说[[宁观]]本身就是[[器灵]]）

#### 【[[春秋戒尺]]】
- **品阶**：道阶中品
- **持有者**：[[颜回风]]（[[白鹿书院]]半圣）
- **外观**：古朴的木质戒尺，上刻春秋大义
- **核心能力**：打神不打人
  - 专门惩戒违背"规矩"者
  - 可以打碎神魂，但不伤肉身
- **来源**：融合了"规矩"属性的[[大道残片]]
- **限制**：只能惩戒"有罪"之人，无法伤害无辜
- **[[器灵]]**：春秋（儒圣的一缕执念）

#### 【[[传国玉玺]]】
- **品阶**：道阶上品
- **持有者**：[[大奉皇帝]]
- **外观**：玉质印玺，底部刻"受命于天，既寿永昌"
- **核心能力**：镇压气运
  - 一印砸下，可镇压[[玉枢境]]修士
  - 在京城内，威力倍增
- **来源**：融合了"气运"属性的[[大道残片]]
- **限制**：只能在大奉疆域内发挥全部威力
- **[[器灵]]**：九州龙魂

### 二、灵阶法宝（宗门重宝）

#### 攻击型

**【[[霜降]]】**（[[本命飞剑]]）
- **品阶**：灵阶下品 → 灵阶中品（进化后）
- **持有者**：[[裴青冥]]（女主）
- **外观**：剑身如霜，寒气逼人
- **核心能力**：
  - 剑气凌厉，攻击无双
  - 可释放冰霜剑气，冻结敌人
- **来源**：[[洗剑池]]赐予的[[本命飞剑]]
- **进化**：[[裴青冥]]突破至[[筑鼎境]]后，[[霜降]]进化为灵阶中品
- **[[器灵]]**：霜（冰冷的少女形象）

**【紫电青霜】**（双剑）
- **品阶**：灵阶极品
- **持有者**：[[李飞白]]（[[太一玄宗]]首席真传）
- **外观**：紫色和青色的双剑，剑身流转雷电
- **核心能力**：
  - 紫电：雷属性，速度极快
  - 青霜：冰属性，冻结万物
  - 双剑合璧：雷霆冰封，威力倍增
- **来源**：[[太一玄宗]]的镇宗之宝
- **[[器灵]]**：紫电和青霜（双生[[器灵]]）

**【[[雷击神木柱]]】**
- **品阶**：灵阶极品
- **持有者**：[[猿魔]]（[[妖王]]）
- **外观**：被雷击过的巨大木柱，重达万钧
- **核心能力**：
  - 力大无穷，可摧毁城墙
  - 蕴含雷霆之力，专克妖魔
- **来源**：上古神木被天雷击中后形成
- **[[器灵]]**：无（[[妖族]]不需要[[器灵]]）

#### 防御型

**【[[黄金算盘]]】**
- **品阶**：灵阶上品
- **持有者**：[[晏三笑]]（[[天机阁]]商人）
- **外观**：黄金打造的算盘，珠子会自动拨动
- **核心能力**：
  - 防御：可展开金色护罩
  - 推演：可推演因果，预测吉凶
- **来源**：[[天机阁]]的传承法宝
- **[[器灵]]**：算（精明的商人形象）

**【[[大荒镇岳盾]]】**
- **品阶**：灵阶极品
- **持有者**：[[铁浮屠]]（[[极寒武帝城]]守将）
- **外观**：半人高的巨盾，由大荒遗种头骨制成
- **核心能力**：
  - 绝对防御：可抵挡上三境攻击
  - 反震：攻击者会被反震之力伤害
- **来源**：击杀大荒遗种后炼制
- **[[器灵]]**：无（武夫不需要[[器灵]]）

#### 辅助型

**【[[招魂红伞]]】**
- **品阶**：灵阶中品
- **持有者**：[[幽姬]]（鬼修）
- **外观**：血红色的油纸伞，伞面绘有鬼魅图案
- **核心能力**：
  - 遮蔽天机：可隐藏持有者的气息
  - 招魂：可召唤亡魂为己用
- **来源**：[[南疆瘴洲]]的鬼修传承
- **[[器灵]]**：无（鬼修本身就是灵体）

**【[[生杀血葫芦]]】**
- **品阶**：灵阶上品
- **持有者**：[[药师琉璃]]（毒师）
- **外观**：血红色的葫芦，一面刻"生"，一面刻"杀"
- **核心能力**：
  - 生：可吐出仙丹，救死扶伤
  - 杀：可喷出化尸水，腐蚀万物
- **来源**：[[百草谷]]的传承法宝
- **[[器灵]]**：生杀（双面人格）

## 第二部分：炼器体系

### 一、炼器师等级

1. **学徒**：可炼制法阶下品法器
2. **炼器师**：可炼制法阶中品法器
3. **大师**：可炼制法阶上品法器
4. **宗师**：可炼制灵阶法宝
5. **天师**：可炼制道阶法宝（极其罕见）

### 二、炼器材料

#### 基础材料
- **玄铁**：最常见的炼器材料，10两[[灵砂]]/斤
- **寒铁**：冰属性材料，50两[[灵砂]]/斤
- **赤铜**：火属性材料，50两[[灵砂]]/斤

#### 高级材料
- **星辰铁**：天外陨石，1枚[[青玉钱]]/两
- **龙鳞**：大荒遗种的鳞片，10枚[[青玉钱]]/片
- **雷击木**：被天雷击中的神木，50枚[[青玉钱]]/斤

#### 顶级材料
- **[[大道残片]]**：融合后可炼制道阶法宝，无价
- **神魔遗骸**：上古神魔的骨骼，无价

### 三、炼器方法

#### 法阶法器炼制
1. **选材**：选择合适的材料
2. **熔炼**：用灵火熔炼材料
3. **塑形**：用神识塑造形状
4. **刻纹**：刻画阵纹，引灵气入物
5. **淬火**：用特殊液体淬火
6. **开光**：注入灵力，激活法器

**时间**：3-7天
**成功率**：70%

#### 灵阶法宝炼制
1. **选材**：必须使用高级材料
2. **去浊**：去除材料中的[[浊气]]
3. **融灵**：融入灵兽精血或天材地宝
4. **刻纹**：刻画复杂的阵纹
5. **[[渡劫]]**：法宝需要[[渡劫]]，才能成为灵宝
6. **养灵**：培养[[器灵]]

**时间**：3-6个月
**成功率**：30%

#### 道阶法宝炼制
1. **获得[[大道残片]]**：这是最难的一步
2. **融合**：将[[大道残片]]融入法宝
3. **承受天罚**：炼制道阶法宝会引来天罚
4. **[[器灵]]觉醒**：道阶法宝必定有[[器灵]]

**时间**：数年
**成功率**：不足10%

## Agent 创作指引

### 法宝获取
1. **宗门赐予**：加入宗门后获得
2. **秘境探索**：在上古遗迹中发现
3. **击杀敌人**：从敌人遗物中获得
4. **购买**：在坊市或拍卖会购买
5. **自己炼制**：需要炼器师技能

### 法宝使用
1. **认主**：法宝需要滴血认主
2. **温养**：法宝需要用灵力温养
3. **损毁**：法宝可以被损毁，需要修复
4. **进化**：法宝可以随主人成长而进化

### 战斗描写
1. **法宝特点**：每个法宝都有独特的能力
2. **克制关系**：火克木，水克火，金克木等
3. **消耗**：使用法宝会消耗灵力
4. **损毁**：激烈战斗中法宝可能损毁
