# 须弥寺

## 基本信息
- **类型**：势力
- **势力等级**：上三境势力
- **创立时间**：[[绝地历]]元年（上古佛国遗留）
- **位置**：[[西极荒漠]]·[[须弥圣山]]
- **住持**：[[空见大师]]（[[玉枢境]]，200岁）
- **成员数量**：约5000人

## 详细描述

### 势力定位
佛门圣地，位于[[须弥圣山]]。以慈悲为怀，普度众生。但也有金刚怒目，降魔卫道。

### 组织架构
- **住持**：[[空见大师]]（[[玉枢境]]）
  - 佛法高深，慈悲为怀
  - 看破红尘，但未看破天道
- **四大金刚**：全部[[寻泉境]]
  1. [[降魔金刚]]（专职降魔）
  2. [[护法金刚]]（守护寺院）
  3. [[传法金刚]]（传授佛法）
  4. [[戒律金刚]]（维持戒律）
- **十八罗汉**：全部[[筑鼎境]]
- **僧侣**：约5000人，下三境到中三境

### 核心理念
**佛门四谛**：
1. 苦谛：世间皆苦
2. 集谛：苦之根源
3. 灭谛：灭苦之道
4. 道谛：修行之法

**慈悲与降魔**：
- 对善者慈悲为怀
- 对恶者金刚怒目
- 降魔卫道，不杀生但也不纵恶

### 核心资源
- **核心功法**：[[大日如来经]]（灵阶极品）
- **[[灵脉]]**：[[地品灵脉]]3条
- **舍利塔**：供奉历代高僧舍利，可净化心魔
- **佛门法器**：[[降魔杵]]（灵阶上品）、[[金刚铃]]（灵阶中品）
- **佛经藏**：收藏上古佛经，记载失传的佛法

### 势力关系
- **[[血海魔宗]]**：死敌，佛魔之争千年
- **[[太一玄宗]]**：互不干涉
- **[[白鹿书院]]**：理念相近，偶有交流
