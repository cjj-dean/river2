# 小说设定集管理系统

> 基于多智能体架构的小说世界观设定管理工具

## 这是什么？

一个智能的小说设定管理系统，帮你：
- 🤖 自动生成详细的设定草案
- 🔍 检测设定之间的逻辑冲突
- 📝 维护一致的世界观体系
- 🔗 通过 MCP 协议与 Claude Code 无缝集成

## 为什么需要它？

写长篇小说时，你是否遇到过：
- ❌ 前后设定矛盾（前面说北方贫瘠，后面又说有三条灵脉）
- ❌ 战力体系崩溃（配角突然比主角还强）
- ❌ 时间线混乱（事件发生时间对不上）
- ❌ 设定文件越来越多，难以管理

这个系统通过 **多个 AI Agent 互相监督**，帮你避免这些问题！

## 工作原理

```
你的需求 → AI 生成草案 → 三个专家审查 → 自动修正 → 写入文件
                           ├─ 战力审查官
                           ├─ 经济审查官
                           └─ 时间线审查官
```

## 5 分钟上手

1. **安装**
   ```bash
   npm install
   ```

2. **配置**
   ```bash
   cp .env.example .env
   # 编辑 .env，填入你的 Anthropic API Key
   ```

3. **测试**
   ```bash
   npm test
   ```

4. **使用**
   - 在 Claude Code 中配置 MCP Server
   - 开始用自然语言管理你的小说设定！

## 示例

**你说**：
> 我想添加一个专门猎杀主角的暗杀组织

**系统做**：
1. 读取你的世界观、流派体系、地图等设定
2. 生成详细的组织设定草案
3. 检查是否与现有设定冲突
4. 自动修正问题
5. 写入对应的设定文件

**结果**：
- ✅ 组织战力符合世界观法则
- ✅ 资源来源合理
- ✅ 历史渊源不冲突
- ✅ 自动更新到相关文件
